Como desenvolvedor web Júnior, uma das minhas principais áreas de expertise é a criação de interfaces interativas, onde utilizo tanto Vue quanto React para criar interfaces interativas e intuitivas.

No entanto, meu conhecimento não se limita apenas ao front-end. Também possuo experiência no desenvolvimento back-end, utilizando os framework sLaravel e Adonis.

Experiência

Desenvolvedor Web (Freelance)

Durante o tempo que trabalhei na Path, tive a oportunidade de integrar a equipe de desenvolvedores responsáveis pelo novo e completo sistema web de gerenciamento escolar.

Utilizando React e TypeScript para front-end e Adonis e Prisma para o back-end. Minha responsabilidade era desenvolver interfaces para os usuários do sistema. Sempre que possível, também contribuía para a criação do back-end.

Esta experiência tem sido uma oportunidade para aprimorar minhas habilidades como desenvolvedor full-stack em JavaScript. Cada dia de trabalho na Path tem contribuído significativamente para consolidar minha trajetória como um desenvolvedor júnior.
